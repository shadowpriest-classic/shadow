About Solace and Insanity (Mind Flay: Insanity)

The Solace and Insanity talent causes your Mind Flay to deal increased damage while Devouring Plague is active on the target. This damage bonus is snapshotted when you start channeling Mind Flay.

Tip: Just before Devouring Plague expires, clip your current Mind Flay channel and start a new one to snapshot the Insanity damage bonus one last time. This maximizes your damage by getting an extra empowered Mind Flay before DP falls off.

⚙️ Update Benchmark Data

Benchmark data is automatically updated weekly via GitHub Actions. To manually update benchmarks now:

Option 1: Run Locally (Recommended)

cd /path/to/shadow
node tools/benchmark-scraper.js --auto

This will fetch ranks 51-100 for all configured encounters and calculate median values. Takes ~2-3 minutes. Requires Node.js installed.

Option 2: Wait for Automatic Update

Next automatic update: Every Wednesday at 3:00 AM CET

Option 3: Trigger GitHub Action

Go to GitHub Actions and manually trigger the "Update Benchmark Data" workflow.
